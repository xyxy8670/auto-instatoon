## Project Identity

Auto InstaToon is a minimal local app for creating 4:5 Korean Instagram carousel comics with Codex OAuth.

## Rules

- Keep the repo small and instatoon-only.
- Do not reintroduce the old broad multi-format pipeline.
- Do not commit `.env.local`, generated images, logs, account tokens, or private source material.
- Prefer direct, readable app code over framework-heavy abstractions.
