# Code of Conduct

## Our Pledge

We want this project to be a practical, respectful place for people building local creator tools with AI.

Participants are expected to:

- be kind and specific in feedback
- assume good intent while still addressing real problems
- respect privacy and creator ownership
- avoid harassment, threats, discrimination, or personal attacks

## Unacceptable Behavior

Unacceptable behavior includes:

- publishing private prompts, files, logs, credentials, or generated client work without permission
- abusive or discriminatory language
- repeated off-topic disruption
- impersonation or misleading claims about project affiliation
- encouraging unsafe use of the software

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block participants who violate this code of conduct.

This document is intentionally concise. For larger community governance, the project may later adopt a fuller contributor covenant.
